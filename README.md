# 工事担任者Pass プライバシーポリシー

最終更新日: 2026年6月29日

公開URL: https://github.com/app-policies/privacy-policy-denki-tsushin-pass-app

工事担任者Pass（以下「本アプリ」）は、電気通信の工事担任者試験の学習を支援する非公式アプリです。本アプリは、電気通信国家試験センター、一般財団法人 日本データ通信協会、総務省、その他の試験実施団体による公式・公認・監修・提携アプリではありません。

本ポリシーでは、本アプリで取り扱う情報、利用目的、第三者サービス、広告、アプリ内課金、トラッキングの有無について説明します。

## 取り扱う情報

本アプリでは、学習機能、広告表示、アプリ内課金、利用状況の分析、品質改善のため、以下の情報を取り扱う場合があります。

- アプリの起動、画面表示、学習開始、回答、学習完了、実戦演習開始・完了、課金画面表示などの利用状況
- 学習モード、対象種別ID、分野ID、回答数、正誤、正答率帯、連続学習日数などの学習状況
- アプリ内課金の購入状態、購入開始、購入完了、復元に関する情報
- 広告配信、広告表示、不正防止、広告効果測定に必要な端末・利用状況に関する情報
- アプリの不具合調査や品質改善に必要な技術情報

## 取得しない情報

本アプリは、以下の情報をアカウント情報として独自に取得しません。

- 氏名
- 住所
- 電話番号
- メールアドレス
- 勤務先
- 受験番号
- 実務経験証明
- 免状申請情報
- 連絡先
- 正確な位置情報

また、Firebase Analytics には以下の情報を送信しません。

- 問題本文
- 選択肢本文
- 解説本文
- 自由入力
- 記述答案
- 個別の問い合わせ内容
- 氏名、メールアドレス、電話番号、住所
- Apple の決済識別子の生値
- 課金管理用の公開キー
- 広告ユニット識別子
- Firebase のアプリ識別子

## 学習データの保存

回答履歴、正誤記録、苦手問題、連続学習日数、文字サイズ、対象種別などの学習データは、原則として利用者の端末内に保存されます。

本アプリは、学習履歴を同期するための独自アカウントサーバーを運営していません。端末の削除、アプリの削除、端末変更、OSの設定変更などにより、端末内の学習データが失われる場合があります。

## Firebase Analytics の利用

本アプリでは、アプリの利用状況を把握し、学習体験や機能品質を改善するために、Google LLC が提供する Firebase Analytics を利用します。

Firebase Analytics では、アプリの起動、初期設定、学習モードの利用、対象種別ID、分野ID、回答の正誤、学習完了、実戦演習完了、スコア帯、課金画面の表示元、レビュー依頼候補など、アプリの利用状況に関するイベントを送信することがあります。

Firebase Analytics には、問題本文、選択肢本文、解説本文、自由入力、記述答案、氏名、メールアドレス、個別相談内容、Apple の決済識別子の生値、広告ユニット識別子を送信しません。

本アプリの初期バージョンでは、Firebase Auth、Firestore、Realtime Database、Cloud Functions、Storage、BigQuery export、Crashlytics、Remote Config、Cloud Messaging、App Check は使用していません。

Firebase Analytics により収集される情報は、Google のプライバシーポリシーおよび Firebase の規約に従って取り扱われます。

- Google プライバシーポリシー: https://policies.google.com/privacy
- Firebase におけるプライバシーとセキュリティ: https://firebase.google.com/support/privacy

## 広告について

本アプリの無料版では、Google Mobile Ads（AdMob）を利用して、バナー広告およびインタースティシャル広告を表示します。

広告配信では、広告表示、広告効果測定、不正防止、配信品質の維持のため、端末情報、広告表示に関する情報、利用状況に関する情報が利用される場合があります。これらの情報の取り扱いは、Google のポリシーに従って管理されます。

初期バージョンでは、広告視聴による問題追加解放は提供していません。

## アプリ内課金について

本アプリでは、非消耗型のアプリ内課金により、買い切りの Premium 機能を提供します。

Premium では、全問題の利用、詳しい解説、選択肢ごとの解説、苦手問題、実戦演習、全問演習、広告非表示などを利用できます。

購入処理は Apple の App 内課金の仕組みを通じて行われ、購入状態の確認、復元、管理には RevenueCat を利用します。RevenueCat では、購入状態の確認や復元に必要な識別子、購入情報が取り扱われる場合があります。

本アプリは、Apple の決済識別子の生値を Firebase Analytics に送信しません。

## 外部送信

本アプリでは、利用状況分析、広告配信、課金管理、品質改善のため、以下の第三者サービスへ情報が送信される場合があります。

- Firebase Analytics
- Google Mobile Ads
- RevenueCat
- Apple が提供する App Store、App 内課金、クラッシュログ、診断関連サービス

各サービスで取り扱われる情報は、それぞれの提供元のプライバシーポリシーおよび利用規約に従って管理されます。

## トラッキングとIDFA

本アプリは、IDFAを使用しません。

本アプリは、Firebase Analytics や広告関連の情報を、他社のアプリやWebサイトをまたいで利用者を追跡する目的では使用しません。そのため、App Tracking Transparency の許可ダイアログは表示しません。

## 公式コンテンツと権利について

本アプリの問題および解説は、公開されている試験概要、試験範囲、試験方式、出題傾向を参考に、スマホ学習向けに独自作成したものです。

本アプリは、公式問題、公開問題、公式正答、公式教材本文、公式図版、公式回路図、公式表、公式ロゴを収録していません。

## 対象外の機能

本アプリは、学習補助を目的としたアプリです。以下の機能は提供しません。

- 工事設計
- 積算
- 施工指導
- 現場診断
- 免状申請代行
- 実務経験証明作成
- 電気通信設備の個別適合判断
- 法的助言
- 行政手続代行
- AI添削
- 外部AI APIを利用した個別回答

試験日程、受験資格、試験方式、免除条件、申請手続、資格者証に関する最新情報は、必ず試験実施団体の案内で確認してください。
