# 工事担任者Pass プライバシーポリシー

制定日: 2026年6月29日

公開URL: `https://app-policies.github.io/privacy-policy-denki-tsushin-pass-app/`

## アプリ

工事担任者Passは、電気通信の工事担任者試験の学習を目的とした非公式アプリです。電気通信国家試験センター、一般財団法人 日本データ通信協会、総務省、その他の公的機関が関与するものではありません。

## 収集する情報

本アプリでは、アプリの利用状況を把握し改善するため、Firebase Analyticsを利用します。送信する情報は画面表示、学習開始、回答、学習完了、購入画面表示などの集計用イベントです。

買い切りPremiumの提供と購入状態の確認のため、AppleのApp内課金およびRevenueCatを利用します。

## 収集しない情報

以下はFirebase Analyticsへ送信しません。

- 問題本文
- 選択肢本文
- 解説本文
- question_id生値
- 氏名
- メールアドレス
- 電話番号
- 個人を直接特定する情報
- 決済ID
- 広告ID
- IDFA
- Firebase App ID
- AdMob ID

## 広告

無料版では広告を表示します。初期v1ではBanner広告とInterstitial広告を利用します。広告リクエストでは非パーソナライズ広告を指定します。広告配信にはGoogle AdMobを利用します。

## 課金

買い切りPremiumを提供します。購入処理はAppleのApp内課金とRevenueCatを利用します。

## IDFA / ATT

本アプリはIDFAを使用しません。トラッキング目的でデータを利用しないため、ATT許可ダイアログは表示しません。

## 公式コンテンツ

公式問題、公開問題、公式解答、公式教材本文、公式図版、公式ロゴは収録しません。問題と解説は学習目的で独自に作成します。
